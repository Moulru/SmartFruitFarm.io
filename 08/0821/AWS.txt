1. AWS 인스턴스 생성
다시 인스턴스를 생성했음. GPU를 사용하는 P타입 G타입 인스턴스유형은 요금이 많이나가기 때문에, AWS자체에서 기본적으로 GPU 인스턴스의 vCPU 할당량을 0으로 제한함.(AWS 정책)
그러므로 P타입을 이용하려면 해당 할당량을 원하는 vCPU만큼 AWS에 증가 요청을 해야하기 때문에 범용 인스턴스를 사용하기로 함.

새로 생성한 인스턴스 타입은 M5.4xlarge (범용 인스턴스타입)

대부분 추천하는 AWS 인스턴스 타입은 M5, C5, R5 임. (사이즈도 대부분 4xlarge 사용)
(M5 : 범용 / C5 : 컴퓨팅 최적화 / R5 : 메모리 최적화)

인스턴스 타입에 대한 설명은 https://aws.amazon.com/ko/ec2/instance-types/ 참고.
